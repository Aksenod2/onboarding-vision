## Appendix 2 — Attributes for CAM (Opening an Account)

> Источник: IMG_6836–6837
> **Attributes for CAM — Opening an Account (4000 or 4500), for Customer with status "Completed":**

| Attribute | Details |
|---|---|
| CIF Number | Auto Generate |
| Customer Name | Auto Populate |
| Customer Type (Corporate) | Auto Populate |
| Branch | Auto Populate |
| Product Name — Current Account | Auto Populate (Editable + Dropdown) |
| Account Class (ex. 4000, 4500) | Auto Populated based on Product Name |
| Currency — INR | Auto Populated (defaulted to INR) |
| CRILC Control — LG / BD / PG account | Radio Button (one to be selected); applicable only if customer selects 4500 |
| Operational Mode for Collection Account | Auto Populated based on exposure entered under CRILC Section |
| Account Description | Auto Populated (same as Customer Name, editable) |
| Account Type (Single or Joint) | Drop Down (fetched from Operating Instructions / Board Resolution / Authority Letter) |
| Mode of Operation | Drop Down (fetched from Operating Instructions / Board Resolution / Authority Letter) |
| CRILC Data | Only if CRILC selected |
| Settlement Instruction | Only if CRILC selected |
| Nominee Info | — |

## Appendix 4 — Documents for Entity

> Источник: IMG_6839–6841
> Таблица показывает по каждому документу: к каким типам юрлиц применим, тянется ли из Probe42, обязателен ли, и как обрабатывается в режимах **STP / Hybrid / Offline**.

| Document | Applicable Entity | Probe42 | Mandatory | STP | Hybrid | Offline | Verification (Hybrid/Offline) |
|---|---|---|---|---|---|---|---|
| Board Resolution / Authority Letter / Management Declaration | Company / LLP / PPF | N | Y | Digital Upload | Digital / Upload | Physical | System / DVU |
| Entity PAN Number | All | Y | Y | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| Certificate of Incorporation | Company & LLP | Y | Y | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| MOA & AOA | Company | Y | Y | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| Share holding pattern | Company | Y | — | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| Company Order of LLP / Partnership Deed | LLP / PPF | Y (LLP) / N (Partnership) | Y | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| Partnership Registration Certificate (if applicable) | LLP / PPF | Y (LLP) / Y (Partnership, if applicable) | N | Auto fetch | Auto fetch / Upload | Physical | System / DVU |
| Address Proof (any valid address proof) | All | Y | Y | Auto fetch | Auto fetch / Upload | Physical | System / DVU |

> ⚠️ Правая колонка **«Additional details / Condition When STP»** в источнике сильно обрезана — условия по каждому документу нечитаемы. Нужен экспорт таблицы из Confluence для полноты.
