## Чек-лист для экрана Board Resolution / подписанты (из письма)

- [ ] Глосса/логика **One single Authorised signatory**
- [ ] **Кто подписывает:** выбор из директоров **ИЛИ** 1 Секретарь
- [ ] **Authorised signatory:** выбор из директоров **+ опция «добавить своего»**
- [ ] Блок **«как в дальнейшем будет проходить смена Authorised signatories»** (выбор из опций)
- [ ] Запрос **контактов** (telephone/email) и для подписывающих, и для AS
- [ ] **MOA/AOA — одно поле** (не два отдельных)

## Чек-лист для дашборда (из письма)

- [ ] Дашборд **со статусами заявки** («что с моей заявкой»)
- [ ] **Drill-down в детализацию** — блок **«Personal Identification & Signing»**: как идёт подписание и VKYC всех участников

## Чек-лист для Aadhaar (из письма)

- [ ] По Aadhaar показываем: name, **aadhaar number маскированный (******XXXX, только последние 4)**, telephone, email, address
- [ ] PAN из Aadhaar НЕ приходит (берётся из Probe / фото в VKYC)
