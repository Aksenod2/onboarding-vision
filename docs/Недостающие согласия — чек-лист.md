## Чего НЕ хватает — достать текст ❌

- [ ] **1. Доступ к данным реестров (consent до PAN)** — экран SP-04.
  - В AS-IS-списке такого текста НЕТ (помечено ⚠ в `Consents — список (current)`).
  - Сейчас в прототипе самописная заглушка с упоминанием Probe42 / PAN-базы — НЕ для показа, заменить на verbatim.
  - Возможно, текст лежит в TO-BE таблице (см. п.3) или покрывается T&C 2b «verify my details digitally» — **уточнить у банка/аналитика**.

- [ ] **2. Декларация Sole Proprietor** (аналог board resolution) — подписывается OTP после VCIP / на шаге `please confirm`.
  - Марго: «тексты в работе», финального нет.
  - Нужен verbatim-текст декларации именно для роли ИП (у ИП нет board resolution).

- [ ] **3. TO-BE набор согласий** (для полностью автоматического онбординга).
  - Отдельная таблица в Confluence: «consents to be taken for the fully-automated onboarding».
  - **Фото пока не снято** → снять и оцифровать (связано: [[Вопросы и расхождения]] В-1).
