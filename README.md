# Сбер — Онбординг юрлиц (1DAY Initiative) · прототип

Кликабельный прототип общего вижна онбординга юрлиц (открытие расчётного счёта, рынок **Индия**) на mock-данных. Боевой фронт на React, который позже забирают разработчики.

> Вынесено из Obsidian-vault `Хранилище/` в самодостаточный репо: спеки + код в одном месте.
> Контекст-указатель в vault: `Хранилище/Сбербанк/Проекты/1D-1 Онбординг клиентов (1DAY Initiative).md`.


## Две роли = две UI-библиотеки (асимметрия)

| Роль | UI | Откуда код |
|---|---|---|
| **Клиент** (заполняет онбординг) | **SDDS Serv** (`@salutejs/sdds-serv`) | боевой опенсорс, MIT |
| **RM** (менеджер банка) | **Caldera** (внутренняя СберДС) | кода нет → свой набор-имитация |
